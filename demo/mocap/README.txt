Extra Features:

- New controls:
  + 'f' key                        - toggles drawing floor
  + Mouse wheel                    - zoom in/out
  + Left/Right arrow keys          - orbit left/right
  + Up/Down arrow keys             - orbit up/down
  + Left mouse button click + drag - orbit around center

- Improved character appearance
  + segments are drawn with elongated spheres
  + wire-frame "tendons" provide illusion of muscle connectivity
  + lighting and shading is enabled
